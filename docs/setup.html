Setup for flash project

Last updated: 2018-02-11

Code version: d53227e

To reproduce our results on your own computer, please follow these setup instructions.

Setup instructions

Install software and R packages

  1. Install R.

  2. Install MATLAB

  3. Install the required R packages (run the following commands in the R).

install.packages(c("devtools", "ashr", "PMA","ssvd","softImpute","Matrix","R.matlab","ggplot2","denoiseR"))
devtools::install_github("stephenslab/flashr")

  1. Install MATLAB package NBSFA

  2. Install packages sfa and SFAmix

Data sets

All the data sets used in the paper are in data files
